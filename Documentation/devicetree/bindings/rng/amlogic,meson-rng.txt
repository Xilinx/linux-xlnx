Amlogic Meson Random number generator
=====================================

Required properties:

- compatible : should be "amlogic,meson-rng"
- reg : Specifies base physical address and size of the registers.

Example:

rng {
        compatible = "amlogic,meson-rng";
        reg = <0x0 0xc8834000 0x0 0x4>;
};
