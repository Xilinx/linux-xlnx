Broadcom BCM23550 device tree bindings
--------------------------------------

This document describes the device tree bindings for boards with the BCM23550
SoC.

Required root node property:
  - compatible: brcm,bcm23550

Example:
	/ {
		model = "BCM23550 SoC";
		compatible = "brcm,bcm23550";
		[...]
	}
